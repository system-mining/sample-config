## Docker configure binary location

**Description**

> To configure to custom binary directory of docker - The default is/usr/local/bin/dockerd
>
> Specs:
>
> * Docker 1.12
> * Ubuntu 16.04

**Config**

```
# sudo vi /etc/default/docker
DOCKERD="your_path_to_dockerd"
```
