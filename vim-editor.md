## Set indent in Vim

**Description:**

> Configure indent in vim
>
> Specs:
>
> * Vim 7.4
> * Ubuntu 16.04

**Config:**

```
# vi ~/.vimrc
set autoindent
set noexpandtab
set tabstop=2
set shiftwidth=2
```
