## Set indent in Vim

**Description:**

> Configure indent in vim
>
> Specs:
>
> * Vim 7.4
> * Ubuntu 16.04

**Config:**

```
# vi /usr/local/apache2/sites-available/vhosts.conf
set autoindent

```
